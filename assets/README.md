# Church Logo Folder

Please upload the church logo here with the following names:
- **church-logo.png** - Main church logo (recommended size: 512x512px)
- **icon.png** - App icon (1024x1024px)
- **adaptive-icon.png** - Android adaptive icon (1024x1024px)
- **splash.png** - Splash screen image (1284x2778px)
- **favicon.png** - Web favicon (48x48px)

The app will automatically use these images once uploaded.
